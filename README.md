# Interactive Scatter Plot (Dash + Plotly)

Simple Dash app to create an interactive scatter plot from `minor_league_advanced.csv`.

## Quick Start

1. Install dependencies:

```bash
python3 -m pip install -r requirements.txt
```

2. Run the app:

```bash
python3 dash_app.py
```

3. Open http://127.0.0.1:8050 in your browser.

Use the dropdowns to select X/Y axes, and optionally color and size by other columns.
